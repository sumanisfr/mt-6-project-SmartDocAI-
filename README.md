# Smart Documentation Platform

A web app to generate technical documentation from uploaded source files or a GitHub repository, with a built-in docs chatbot.

## Features
- Upload files or add a GitHub repository URL
- Generate structured markdown documentation
- Browse sections in a docs viewer
- Ask questions based on generated docs

## Tech Stack
- Backend: Node.js, Express
- Frontend: React, Vite, Tailwind CSS
- AI: xAI Grok API

## Run Locally
1. Install dependencies:
   - backend: npm install
   - frontend: npm install
2. Start backend:
   - cd backend
   - npm start
3. Start frontend:
   - cd frontend
   - npm run dev

## API
- Health: /health
- Project operations: /api/projects
